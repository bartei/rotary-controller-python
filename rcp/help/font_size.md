Default Font Size
=================

The font size (in pixels) used for the DRO text fields and
position readouts on the home screen.

## Guidelines

- Adjust based on your screen size and how many axes you display
- Typical range: 10–20 px

## Recommendations

| Screen Size     | Suggested Font Size |
|-----------------|---------------------|
| 7" touchscreen  | 14                  |
| 10" tablet      | 16–18               |
| Desktop monitor | 18–20               |

## Notes

- Value must be a positive integer
- Changes take effect immediately on the home screen
- The default value of 14 works well for a standard 7" Raspberry Pi
  touchscreen
